    <h1 id="markdown-header-geforce-auto-update">geforce-auto-update</h1>
<p>Checks for updates to nVidia video drivers. Latest version is retrieved from Chocolatey package page.
https://chocolatey.org/packages/geforce-game-ready-driver-win10</p>
<p>If new updated is found user is prompted for install mode.</p>
<p>Manual: Opens direct download like for the driver in default browser.</p>
<p>Automatic: Performs the driver install automaticaly (user still gets UAC prompts). Default installation progress window is still shown but no interaction is required.</p>
<p>Either place shortcut to the Startup folder or use Windows Task Manager to run periodicaly.
C:\Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup</p>
